Scopo¶

Il Plugin Lidar Tile Maker Plugin è stato sviluppato per rendere più semplice l’utlizzo del Plugin CHM from LIDAR. Quest ultimo è stato sviluppato da Gter srl e finanziato dal “Programma di sviluppo rurale per il Veneto 2014-2020” - Regione del Veneto - Direzione AdG FEASR e Foreste. Permette di calcolare il Canopy Height Model (CHM) a partire da dati LIDAR, in particolare Digital Surface Model (DSM) e Digital Terrain Model (DTM).

L’utilizzo del Plugin CHM from LIDAR si basa su un Layer vettoriale di Tile che viene automaticamente caricato nel progetto Qgis corrente premendo l’icona del plugin Di default il layer vettoriale delle tile, nominato tile_dsm_dtm.gpkg, contiene i dati della Regione Veneto che ne ha finanziato lo sviluppo. Ovviamente il layer tile_dsm_dtm può essere modificato dall’utente inserendo i propri dati.

Il Plugin Lidar Tile Maker permette all’utente di creare il proprio Layer Vettoriale di Tile con tutte le caratteristiche richieste dal Plugin CHM from LIDAR Per esempio, il plugin crea un file tile_dsm_dtm.gpkg con tutte le informazioni necessarie immagazzinate nella tabella degli attributi. I campi della tabella sono creati automaticamente e nominati in modo da poter essere utilizzati dal Plugin CHM from LIDAR.

L’output finale è un file GeoPackage (tile_dsm_dtm.gpkg).

Glossario¶

  • CHM: Canopy Height Model

  • DSM: Digital Surface Model

  • DTM: Digital Terrain Model

  • GIS: Geographic Information System

  • LIDAR: Laser Imaging Detection and Ranging
